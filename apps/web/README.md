# apps/web — Frontend Dashboard

**Status: Phase 1 scaffold complete.**

This directory contains the minimal Next.js 15 frontend scaffold. The current home page
is a Phase 1 placeholder only; it does not fetch data or implement the review inbox.

## Planned stack
- Next.js 15 (App Router)
- TypeScript (strict mode)
- Tailwind CSS
- Supabase JS client (future phase)

## Role in the system
The frontend is the review and control layer. Its primary job is not to display live feeds
of data — it is to show the user what the AI has captured and classified, and give them
the tools to confirm, edit, or reject each item before it becomes a permanent record.

The home screen will become the **review inbox** in Phase 5.
