# apps/web — Frontend Dashboard

**Status: Not yet scaffolded. Reserved for Phase 1.**

This directory will contain the Next.js 15 frontend dashboard.

## Planned stack
- Next.js 15 (App Router)
- TypeScript (strict mode)
- Tailwind CSS
- Supabase JS client

## Role in the system
The frontend is the review and control layer. Its primary job is not to display live feeds
of data — it is to show the user what the AI has captured and classified, and give them
the tools to confirm, edit, or reject each item before it becomes a permanent record.

The home screen is the **review inbox**: a list of pending classified items waiting for
human confirmation.
